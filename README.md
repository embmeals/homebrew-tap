# homebrew-tap

Homebrew formulae for [embmeals](https://github.com/embmeals) packages.

## Install

```bash
brew tap embmeals/tap
```

## Formulae

| Formula | Description |
|---|---|
| [terminal-dashboard](https://github.com/embmeals/terminal-dashboard) | Terminal-based media server status dashboard |

```bash
brew install terminal-dashboard
```
